VSCode 使用指南及快捷键

1. 什么是VSCode？

VSCode（Visual Studio Code）是一个由微软开发的轻量级、强大的源代码编辑器，支持多种编程语言，集成了Git版本控制，拥有强大的扩展功能，适用于开发、调试和测试代码。

2. VSCode的主要功能

  • 多语言支持: 支持JavaScript、Python、Java、Go、C++等多种语言。
  • 扩展市场: 提供丰富的扩展插件，帮助定制开发环境。
  • 集成终端: 内置终端，可以在编辑器中直接运行命令。
  • 调试器: 支持直接调试代码，查看变量和堆栈信息。
  • Git集成: 提供Git操作界面，方便查看和提交代码变更。
  • 智能提示: 提供代码补全、语法检查等功能。

3. VSCode的安装

VSCode可以从官方站点下载并安装，支持Windows、macOS和Linux操作系统。

4. VSCode的基本操作

4.1 打开文件夹/项目

点击文件 -> 打开文件夹，选择你要编辑的项目文件夹。

4.2 安装扩展插件

在左侧活动栏中点击扩展图标（小方块），然后搜索需要的扩展，点击“安装”按钮。

4.3 调试程序

点击左侧活动栏中的调试图标，设置断点，选择调试配置并启动调试。

4.4 终端使用

按下快捷键Ctrl + （反引号），即可打开内置终端，运行命令。

5. 常用快捷键

5.1 文件和编辑

  • Ctrl + P: 快速打开文件。
  • Ctrl + N: 新建文件。
  • Ctrl + S: 保存文件。
  • Ctrl + W: 关闭当前文件。
  • Ctrl + Shift + S: 另存为。
  • Ctrl + Shift + T: 恢复关闭的文件。
  • Ctrl + X: 剪切当前行。
  • Ctrl + C: 复制当前行。
  • Ctrl + V: 粘贴。
  • Ctrl + Z: 撤销操作。
  • Ctrl + Shift + Z: 重做操作。
  • Ctrl + /: 单行注释/取消注释。
  • Ctrl + Shift + A: 块注释/取消注释。
  • Alt + Up/Down: 上下移动当前行。

5.2 多光标与选择

  • Alt + Click: 添加多个光标。
  • Ctrl + D: 选择单词，并添加下一匹配项到选择。
  • Ctrl + Shift + L: 选择所有匹配项。
  • Shift + Alt + Up/Down: 向上或向下添加光标。
  • Ctrl + Shift + K: 删除当前行。

5.3 导航

  • Ctrl + T: 查找符号。
  • Ctrl + G: 跳转到某行。
  • Ctrl + Shift + O: 跳转到文件中的符号。
  • Ctrl + F: 查找。
  • Ctrl + H: 查找并替换。
  • F12: 跳转到定义。
  • Alt + Left/Right: 在编辑器中向前/向后导航。

5.4 调试

  • F5: 启动/继续调试。
  • Shift + F5: 停止调试。
  • F9: 切换断点。
  • F10: 单步跳过。
  • F11: 单步进入。
  • Shift + F11: 单步跳出。

5.5 终端

  • `Ctrl + ``: 打开内置终端。
  • `Ctrl + Shift + ``: 新建终端。
  • Ctrl + C: 终止命令。

5.6 Git快捷键

  • Ctrl + Shift + G: 打开Git视图。
  • Ctrl + Shift + G + Ctrl + Enter: 提交变更。
  • Ctrl + Shift + G + Ctrl + D: 丢弃更改。

6. VSCode常用扩展插件推荐

  • Prettier: 代码格式化工具，支持多种语言。
  • ESLint: JavaScript和TypeScript代码规范检查工具。
  • Python: Python开发扩展，提供调试、代码提示等功能。
  • GitLens: 提供更强大的Git可视化工具，查看提交历史、差异等。
  • Live Server: 开发HTML页面时的实时预览工具。

7. 自定义快捷键

你可以通过以下方式自定义自己的快捷键：

  1. 打开VSCode后，点击文件 -> 首选项 -> 键盘快捷方式。
  2. 在搜索框中输入想要修改的命令。
  3. 右键单击并选择“更改键绑定”以自定义快捷键。

VSCode是一款功能强大且易于扩展的编辑器，适用于前端、后端和全栈开发人员。通过学习并熟悉其快捷键，可以大大提升开发效率。
